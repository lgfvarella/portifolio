# Como executar o projeto

## Back end
Pré-requisitos: nenhum

## Front end web
Pré-requisitos: npm / angular version 16

```bash
# clonar repositório
git clone https://github.com/lgfvarella/portifolio.git

# entrar na pasta do projeto front end web
cd portifolio

# instalar dependências
npm install

# executar o projeto
ng serve --open
```
